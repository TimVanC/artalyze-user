Artalyze - BETA Version 1.2 Patch Notes

Release Date: February 25th, 2025

New Features & Improvements

Image Loading & Stability Enhancements

  • Preloaded Enlarged Images: Enlarged images now load instantly instead of reloading every time they are viewed.
  • Persistent Image Order: The randomization of human and AI images is now stored in localStorage, preventing images from switching positions after a refresh.
  • Loading Indicators: Implemented a loader over image containers while images are loading, ensuring better feedback on slower connections.

Bug Fixes & Optimizations

Image Pair Randomization & Flickering Fixes

  • Fixed Position Switching on Refresh: Image pairs now maintain their initial order throughout the puzzle, eliminating flickering and re-randomization on reload.
  • Cached Image Order: The daily puzzle is now saved in localStorage, ensuring that once an image order is set, it remains unchanged until the next puzzle.

Submit Button Activation Fixes

  • Submit Button Only Activates When All Pairs Are Selected: Fixed an issue where selecting only the last image incorrectly enabled the submit button.
  • Deselecting an Image Properly Disables the Submit Button: If a selection is removed, the submit button will now deactivate unless all pairs remain selected.
  • Clearing Selections Resets Button State: Pressing "Clear" correctly disables the submit button and resets the selection state.

Artalyze - BETA Version 1.1 Patch Notes

Release Date: February 17th, 2025

New Features & Improvements

Swipe Overlays for New Players

  • First-time users now see a “Swipe right” overlay after their first selection.
  • After swiping, a “Swipe left to go back” overlay appears to guide navigation.
  • A “Double tap to enlarge image” overlay now appears after the second swipe.
  • After the fourth swipe, a “Tap info icon for more help” overlay displays.
  • These overlays only appear once per user to maintain a seamless experience.

Dark Mode Overlay Enhancements

Overlays now have a lighter background with black text in dark mode for improved readability.

Mid-Turn Overlay Feedback Improvement

Players now receive clearer, more specific feedback after submitting incorrect guesses:

  • 0-1 correct: "None or only one is right. Rethink your picks."
  • 2 correct: "You're three away."
  • 3 correct: "You're two away."
  • 4 correct: "Close! You're one away."

This provides better guidance and allows players to adjust their strategy accordingly.

Bug Fixes & Optimizations

  • Ensured overlays only trigger once per user and do not reappear after being dismissed.
  • Fixed an issue where handleSwipe was not correctly tracking first-time swipes.
  • Resolved a bug causing overlays to reappear on subsequent swipes.
  • Adjusted overlay spacing for improved readability and reduced visual clutter.
  • Minor UI adjustments to keep overlays compact and non-intrusive.
